## CACHE MANIFEST
###### 在此标题下列出的文件将在首次下载后进行缓存
~~~
<html manifest="demo.appcache">
~~~
## # VERSION
~~~
修改版本号可刷新缓存
~~~

## NETWORK
###### 在此标题下列出的文件需要与服务器的连接，且不会被缓存
~~~
.可以使用星号来指示所有其他资源/文件都需要因特网连接：
NETWORK:
*
~~~
## FALLBACK
###### 在此标题下列出的文件规定当页面无法访问时的回退页面（比如 404 页面）
~~~

----------------------
FALLBACK:
/html5/ /404.html
~~~
